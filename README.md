# Music Compilation
Just a small app I made while learning Spring framework.

Used Spring Boot & Thymeleaf.

Download the mysql dump file from sql folder and execute in Mysql Workbench

Run the app in cmd: mvnw spring-boot:run

http://localhost:8585
